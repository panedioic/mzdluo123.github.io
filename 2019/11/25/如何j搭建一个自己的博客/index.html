如何搭建一个像这样的静态博客

准备工具

  • 一个GitHub账号
  • 搜索引擎
  • git客户端
  • 你的大脑

第一步：创建博客仓库

在GitHub创建一个仓库，其中仓库名称必须为你的用户名.github.io，仓库类型为公开仓库

安装博客工具

首先安装nodejs，这一步请自行百度安装

完成后，打开命令行输入npm install -g hexo-cli，等待安装完成

你还需要安装一个插件

输入hexo d来部署你的博客到GitHub，也可以使用hexo s开启本地测试服务器
